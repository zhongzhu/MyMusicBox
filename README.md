MyMusicBox
==========

MyMusicBox is created to learn how to code a sample project called "MyMusic" with using Sencha Touch 2.4.0.

I've put this project on website at http://mymusicbox2.sinaapp.com , check it out.

From this sample project, I learned
* Tab view
* Navigation view
* Nested List
* ActionSheet
* Tree Store
* use Audio class to play mp3
* CRUD on Web SQL
* How to call another contoller from one controller
